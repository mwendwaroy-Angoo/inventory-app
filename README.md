Inventory App — Supply Chain Enhancements

This repository is an inventory and marketplace application. Recent enhancements add simple supply-chain features:

- Demand-driven reorder recommendations (lead-time demand, safety stock, reorder point)
- `PurchaseOrder` and `PurchaseOrderLine` models to track on-order quantities
- Owner UI to create/edit purchase orders with inline lines
- Management command to generate recommendations

See `docs/purchase_orders.md` for details and usage instructions.
